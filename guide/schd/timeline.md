# 设置时间线
本文将为您介绍如何在 Class Widgets 中设置时间线。
## 介绍
时间线规定了在一天中各个节点和活动时长，如课程、课间等。Class Widgets 将根据时间线及当前时间显示课程表中对应的课程。

下面，请根据您使用的软件版本配置。
## v1.1.7 或更新版本 <Badge type="warning">beta</Badge> {#v117-or-newer}
::: danger
只适用于 `v1.1.7` 或更新版本  
对于 `v1.1.6.5` 或更早版本，参见[这里](#v1165-or-older)
:::
